# Mathematical Function Graph Analysis Tool

A web-based mathematical function visualization and analysis tool.

## Features Overview

An interactive mathematical function graph analysis tool supporting the following core features:

- Input and editing of polynomial coefficients
- Real-time plotting and visualization of function graphs
- Analysis and display of function monotonicity
- Growth rate analysis
- Critical point calculation
- Dual-function comparison mode

## Usage Instructions

### Basic Operations

1. Enter polynomial coefficients in the left control panel (supports quadratic and higher-order polynomials)
2. Observe the function graph updating in real time on the right
3. Review the analysis results below: derivative formula, monotonic intervals, growth rate, and critical points

### Interface Layout

- **Left Panel**: Coefficient input, custom function toolbar, range settings, comparison mode toggle
- **Central Area**: Function graph visualization canvas (drag to resize)
- **Right Panel**: Formula display, derivative formula, monotonicity analysis, growth rate, critical points

### Comparison Mode

Click the comparison button on the interface to expand a second set of function inputs and analysis results, enabling easy comparison of two different functions' properties.

## Technology Stack

- Native JavaScript
- Canvas API (graph rendering)
- CSS3 (responsive layout)

## Browser Support

Recommended browsers for optimal experience: Chrome, Firefox, Edge, Safari.
