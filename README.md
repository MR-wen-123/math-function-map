# 函数单调性 · 动态图谱

基于浏览器的数学函数可视化与分析工具，支持 f(x) / g(x) 对比、符号求导、单调性与增减快慢分析，以及图像上的交互探针与控制点。适合复合函数、分式、根式与绝对值等高中常见题型。

[English documentation](README.en.md)

## 功能概览

- **多种函数类型**：一次、二次、三次、指数、对数、幂函数，以及自定义表达式
- **系数编辑**：支持整数、小数与分数（如 `1/2`、`-3/4`、`1 1/2`）；图像上可拖动控制点改参数
- **自定义表达式**：`sin`、`cos`、`tan`、`ln`、`log`、`sqrt`、`abs`、`exp`、常数 `pi` / `e`；隐式乘法如 `2x`、`x(x-1)`、`2sin(x)`；符号快捷输入栏与**例题**一键填入
- **定义域推断**：分式奇点、`√(二次式)`、`ln` 等自动给出区间并用于作图与分析（如 `√(x²−1)` → (−∞,−1]∪[1,+∞)）
- **定义域限制**：f(x)、g(x) 可分别设置左/右端点（含 `±∞`）
- **双函数对比**：启用 g(x) 后同图绘制两条曲线，并分别显示公式、导数、单调性与增减快慢
- **符号求导**：自定义式在可解析范围内给出 f′(x) / g′(x)（含 `|二次|` 分段、部分复合式）；复杂式可能回退为数值说明
- **图像分析**：单调区间着色、导数曲线、增减快慢（速率着色）、临界点；高频振荡函数自动压缩区间说明并减少标注
- **显示开关**：可单独开关关键点、探针/切线、控制点
- **画布交互**：滚轮缩放（以光标为中心）、空白处拖拽平移、单击取点与切线、**回到原点**恢复默认视图；高精度视窗与自适应刻度
- **坐标标注**：可开关图像上的坐标文字
- **布局调节**：左右侧栏宽度、图像区域高度；三栏圆角与栏间距；侧栏与图区可收起
- **公式排版**：分析区函数式与导数使用内置 [KaTeX](https://katex.org/) 渲染，**无需外网 CDN**

## 快速开始

克隆或下载后，若仓库内已有 `vendor/katex/`，可直接打开页面；否则先安装依赖：

```bash
npm install
# 将 KaTeX 复制到 vendor/katex/（npm install 时会自动执行）
```

### 方式一：多文件版（开发）

用浏览器打开项目根目录下的 `index.html`（需与 `vendor/katex/` 同目录），或通过本地静态服务器访问。

```bash
# 示例：Python 内置服务器
python -m http.server 8080
# 浏览器访问 http://localhost:8080/index.html
```

### 方式二：单文件版（分发）

直接双击或用浏览器打开 `standalone.html`，无需服务器；**KaTeX 与字体已内嵌**，可离线使用。

从多文件版重新生成单文件（需先 `npm install`）：

```bash
npm run build
```

## 自定义表达式说明

| 写法 | 含义 |
|------|------|
| `x**2` 或 `x^2` | x² |
| `x(x-1)` | x·(x−1)（非“调用 x”） |
| `1/(x(x-1))` | 1/(x²−x) |
| `log(x)` | 常用对数 log₁₀(x) |
| `ln(x)` | 自然对数 |
| `exp(x)`、`e**(sqrt(x))` | 指数 |

在 **自定义 f(x)** 下，左侧 **例题** 提供复合函数常见题型（如 `√(x²−1)`、`|x²−2x−3|`、`1/(x(x−1))` 等），点击即可填入；悬停按钮可查看简要说明。

## 界面说明

| 区域 | 内容 |
|------|------|
| **左侧 · 参数设置** | 函数类型、系数、自定义式、例题、定义域、对比 g(x)、视图范围、显示开关 |
| **中部 · 函数图像** | Canvas 绘图；底部横条调节高度；可收起图区 |
| **右侧 · 分析结果** | KaTeX 公式与导数、定义域、单调性、增减快慢、临界点；可收起 |

**桌面端（宽屏）**：三栏横排。  
**窄屏（≤1100px）**：上图、中控、下分析；在图像上按住 **Ctrl**（Mac：**⌘**）再滚轮可缩放。

## 图像操作

### 滚轮与页面滚动

| 场景 | 在图像上滚轮 | 在图像上 Ctrl / ⌘ + 滚轮 |
|------|----------------|---------------------------|
| 窄屏（≤1100px） | 滚动页面 | 缩放视图 |
| 宽屏且页面比视口高 | 滚动页面 | 缩放视图 |
| 宽屏且一屏能放下全部内容 | 缩放视图 | 缩放视图 |

光标在图像内时，缩放中心跟随光标位置。

### 其他操作

1. **拖拽**（空白区域）：平移坐标系
2. **单击**：设置探针，查看该点 f/g 值、切线与增减快慢
3. **拖动控制点**（曲线上的手柄）：修改对应系数
4. **回到原点**：恢复默认视图范围
5. **坐标标注**：开关曲线上坐标与标注文字

## 技术栈

- 原生 HTML / CSS / JavaScript
- Canvas 2D API
- 自研表达式 AST 解析、定义域推断与符号求导（`math-engine.js`）
- KaTeX 0.16（`vendor/katex/`，单文件版内嵌）

## 浏览器支持

推荐使用最新版 Chrome、Edge、Firefox、Safari。需支持 Canvas、`matchMedia`，以及可选的 `ResizeObserver` 与 `visualViewport`。
