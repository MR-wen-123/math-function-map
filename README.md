


# 数学函数图像分析工具

A web-based mathematical function visualization and analysis tool.

## 功能简介

一个交互式的数学函数图像分析工具，支持以下核心功能：

- 多项式函数的系数输入与编辑
- 函数图像实时绘制与可视化
- 函数单调性分析与显示
- 增长率分析
- 临界点计算
- 双函数对比分析模式

## 使用说明

### 基本操作

1. 在左侧控制面板中输入多项式系数（支持二次及更高次幂）
2. 观察右侧实时更新的函数图像
3. 查看下方的分析结果：导数公式、单调区间、增长率和临界点

### 界面布局

- **左侧面板**：系数输入、自定义函数工具栏、范围设置、对比模式切换
- **中部区域**：函数图像可视化画布（可拖拽调整大小）
- **右侧面板**：公式显示、导数公式、单调性分析、增长率、临界点

### 对比模式

点击界面上的对比按钮，可展开第二组函数输入和分析结果显示，方便比较两个不同函数的性质。

## 技术栈

- 原生 JavaScript
- Canvas API（图像绘制）
- CSS3（响应式布局）

## 浏览器支持

推荐使用现代浏览器（Chrome、Firefox、Edge、Safari）以获得最佳体验。
